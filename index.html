Big O Notation

What is Big O Notation?

Big O Notation is used in computer science to describe ... the execution time as well as the storage space a function will take. It is written as O(n) with n describing the efficiency of the function.

Constant Time: O(1)

Constant time is when a function takes a fixed time to run and the run time is independent from the value. 

public int returnNum(int num) {   
   return num * num + nums;   
}

Constant time takes the same amount of time no matter what value is passed through the function. In Big O, it is written as O(1).

Linear Time: O(n)

Linear time indicates a function will execute n amount of times. The efficiency of a linear time function would be O(n). 

public int returnNum(int num) {   
   int n = 0;   
   for(int i = 0; i < num ; i++) {   
      n += i ;   
   }   
   return n ;   
}
